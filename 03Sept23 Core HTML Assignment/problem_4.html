1. <html>: The root element that encapsulates the entire HTML document.
  2. <head>: Contains meta-information about the HTML document, such as title, metadata, stylesheets, and scripts.
  3. <title>: Defines the title of the HTML document displayed in the browser's title bar or tab.
  4. <body>: Encloses the content visible on the web page, including text, images, links, and other elements.
  5. <h1> to <h6>: Heading tags representing hierarchical levels of headings, where h1 is the most important and h6 is the least important.
  6. <p>: Defines a paragraph, used to structure text content within the document.
  7. <a>: Creates hyperlinks to other web pages, files, locations within the same page, or email addresses.
  8. <img>: Embeds images into the HTML document, displaying graphics on the web page.
  9. <ul>: Stands for "unordered list" and creates a bulleted list of items.
  10. <ol>: Stands for "ordered list" and creates a numbered or ordered list of items.
  11. <li>: Used within <ul> or <ol> to define list items, whether bulleted or numbered.
  12. <div>: Represents a division or a container that helps in organizing and grouping elements for styling and layout purposes.
  13. <span>: Similar to <div>, but typically used for inline elements to apply styles or manipulate content within a line of text.
  14. <table>: Creates a structure for displaying tabular data on a web page.
  15. <form>: Encloses form elements like input fields, buttons, checkboxes, etc., used for user input and interaction.
